# CHONK – Obsidian Vault (Team AngryPike)

Wissens- und Dokumentations-Vault für **CHONK** (mythologischer Friendslop-Combat-Racer), Modul *Game Project 4* (SoSe 2026), HNU.

## Schnellstart

1. **Obsidian** installieren: https://obsidian.md
2. Diesen Ordner per **„Open folder as vault"** öffnen.
3. Startseite: `00_Start/🏠 Home.md`.
4. Team-Sync über das **Obsidian-Git-Plugin** einrichten → siehe `00_Start/Vault-Anleitung.md`.

## Struktur

| Ordner | Inhalt |

## Quellenprinzip

Projekt-interne Fakten sind an Teamdokumente gebunden (z. B. *GDD, Gehrke, 2026*). Externe Aussagen sind nach **APA** belegt (`07_Quellen/Quellenverzeichnis_APA.md`); es werden ausschließlich frei zugängliche Quellen verwendet.
