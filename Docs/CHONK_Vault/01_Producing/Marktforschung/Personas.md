# Personas

Drei aus der Review-Auswertung abgeleitete Personas ([[Wettbewerbsanalyse_Review-Auswertung]]).

## A — Kompetitiver Arcade-Racer

| Merkmal | Wert |
|---|---|
| Alter | ~20–35 |
| Plattform | PC / Konsole |
| Session | 45–90 Min, 3–5×/Woche |
| Motivatoren | Skill-Expression, faire Belohnung, taktische Tiefe |
| Frust | Rubber-Banding, unbrauchbare KI-Steuerung, Pay-to-Win |
| Monetarisierung | akzeptiert Kosmetik; lehnt **Pay-to-Win** ab |
| Zahlungsbereitschaft | ~15–30 € einmalig + kosmetische DLCs |

## B — Casual Mobile-/Social-Spieler

| Merkmal | Wert |
|---|---|
| Alter | ~25–40 |
| Plattform | Mobile / PC |
| Session | 5–20 Min, täglich kurz |
| Motivatoren | unkompliziertes Einsteigen, sozialer Kontext, kurze Sessions |
| Frust | steile Lernkurve, komplexe Steuerung, Überforderung |
| Monetarisierung | F2P mit Kosmetik; lehnt verwirrende Währungen & FOMO ab |
| Zahlungsbereitschaft | 0–5 € / Monat im F2P-Kontext |

## C — Nischen-Enthusiast

| Merkmal | Wert |
|---|---|
| Alter | ~20–30 |
| Plattform | PC / Konsole / Retro |
| Session | 30–60 Min, 1–2×/Woche |
| Motivatoren | einzigartiges historisches Setting, Mythologie, Sammeln |
| Frust | Setting-Brüche, Grind ohne Narrativ |
| Monetarisierung | Einmalkauf + kosmetische DLCs; lehnt P2W & setting-fremde Inhalte ab |
| Zahlungsbereitschaft | ~10–20 € einmalig (Vollpreis 50 € kritisch) |

## Implikation

CHONK adressiert primär **A** (Combat-Tiefe, faires Balancing) und **C** (Antike-Setting, Götter, Sammelbares Kosmetik), bleibt durch niedrige Lernkurve aber für **B** zugänglich. Das stützt den Premium-Pivot ([[05_Marktforschung/Geschäftsmodell_Preisstrategie]]) und das kosmetik-basierte Monetarisierungsdesign (keine P2W-Mechanik im Chimären-/Charakter-Shop).
