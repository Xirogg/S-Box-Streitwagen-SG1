# Namenskonvention (Assets)

Alle Dateien werden **ausnahmslos auf Englisch** benannt (Toolkompatibilität, teamübergreifende Zusammenarbeit) (TAD, Felsing, 2026).

## Schema

```
[Type]_[Name]_[Specification]_[Number].[Ending]
```

Spezifizierungen (Ausrichtung, Variante) stehen zwischen Name und Nummer und werden bei Bedarf angegeben.

**Beispiele:**
- `A_Horse_01.fbx`
- `A_Chariot_Wheel_L.fbx`
- `T_Chariot_Albedo_02.png`

## Type-Präfixe

| Präfix | Bedeutung |
|---|---|
| `A` | Asset |
| `AN` | Animation |
| `T` | Texture |
| `M` | Material |
| `S` | Sound |
