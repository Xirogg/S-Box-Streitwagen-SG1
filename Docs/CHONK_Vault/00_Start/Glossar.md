# 📖 Glossar — Projektbegriffe CHONK

Zentrale Begriffe quer durch GDD, Tech Bible, TAD und Producing-Bibles. Quelle jeweils in Klammern.

| Begriff | Bedeutung |
|---|---|
| **CHONK** | Arbeitstitel des Spiels (mythologischer Friendslop-Combat-Racer). (GDD, Gehrke, 2026) |
| **AngryPike** | Name des Entwicklungsteams. |
| **Chohonk** | Rundlicher, kugelförmiger Spielercharakter. (Tech Bible, Lange, 2026) |
| **Chimäre** | Zugtier; zwei Chimären ziehen einen Streitwagen. Austauschbar wie Skins (Baukasten-Prinzip, mit Stats). (GDD, Gehrke, 2026) |
| **Friendslop** | Casual-Multiplayer-Subgenre, das durch soziales Chaos lebt; 2023–2025 sehr erfolgreich, bislang horror-dominiert. (Friendslop Research Bible, Würfl, 2026) |
| **Publikumsgunst** | Spielwährung. *Verfügbar* = gesammelt − ausgegeben; *Erfahrung* = gesamte je gesammelte Gunst. (GDD, Gehrke, 2026) |
| **Götter-Gunst** | Am Opfer-Altar erkaufter Vorteil fürs nächste Rennen (garantierte Normal-Fähigkeit oder erhöhte Chance auf Götter-Fähigkeit). (GDD, Gehrke, 2026) |
| **Götter-Fluch** | Strafe bei Gier (mehr als eine Gunst gekauft): 50 % Chance je weiterer Gunst, Verlust der Vorteile + Kopfgeld. (GDD, Gehrke, 2026) |
| **Kopfgeld** | Markierung eines Spielers; andere erhalten mehr Gunst, wenn sie ihm Schaden zufügen. (GDD, Gehrke, 2026) |
| **Opfer-Altar** | Meta-Station: Gunst gegen Götter-Gunst eintauschen (roguelike-artiges Zufallsangebot). (GDD, Gehrke, 2026) |
| **Omen / Gunst-Zone** | Durchfahrbare Zone im Rennen, die ein passendes Power-Up gibt; Zurückliegende erhalten schneller/stärker Gunst (Rubberbanding). (Game-Loop Langformat, 2026) |
| **Ruck-Manöver (Q/E)** | Tap-basierter seitlicher Ausbruch der Chimären — dient Combat **und** engem Kurvenfahren. (GDD, Gehrke, 2026) |
| **Wucht** | Kollisionskennwert: `Gewicht × aktuelle Geschwindigkeit`. (GDD, Gehrke, 2026) |
| **Fliehkraft** | `F = m · (v² / r)` — Grundlage des Kurvenverhaltens. (GDD, Gehrke, 2026) |
| **Random Event** | Zufälliger Göttereingriff pro Checkpoint/Spieler (z. B. herabfallende Riesentraube von Dionysos). (GDD, Gehrke, 2026) |
| **Vision-Anker** | Fünf Leitkriterien für Design-Reviews. (Progress Update Woche 4, Würfl, 2026) |
| **DoD** | *Definition of Done* — drei Stufen: Prototype / Sprint / Done-Done. (QM-Strategie Research Bible, Würfl, 2026) |
| **RACI** | Responsible · Accountable · Consulted · Informed — Zuständigkeitsmatrix. (Research Bible Producing, Würfl, 2026) |
| **s&box** | Source-2-basierte Engine von Facepunch; nutzt **Component-System** (nicht das Legacy-Entity-System), C#. (Tech Bible, Lange, 2026) |
