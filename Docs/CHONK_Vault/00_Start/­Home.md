## 🎯 Design Pillars / Vision-Anker

Fünf Vision-Anker für jedes Design-Review (Progress Update Woche 4, Würfl, 2026):

1. **Geschwindigkeit ist spürbar** — FOV-Weitung, Motion-Blur, SFX
2. **Götter-Fähigkeiten als Equalizer** — Comeback-Mechanik
3. **Antike als Setting, nicht als Simulation** — stilisiert, lesbar
4. **Sozial vor Solo** — jede Entscheidung priorisiert Multiplayer
5. **5-Minuten-Rennen** — spielbar in einer Mittagspause

→ Details: [[02_GameDesign/Vision_und_Pillars]]
