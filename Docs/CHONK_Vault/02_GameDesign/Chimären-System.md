# Chimären-System

Ein „Fahrzeug" besteht aus **2 Zugtieren (Chimären)** + einem **Streitwagen** (GDD, Gehrke, 2026).

- Chimären-Teile werden **wie Skins** getauscht und basieren auf **demselben Grundmodell**.
- Ziel: **abstrakte Kombinationen** ermöglichen → bewusste *Chaos-Maximierung*.
- Im **Chimären-Stall** (Meta-Station) gekauft/ausgewählt — **kosmetisch**. Die taktisch relevanten Stats liegen am **Wagen** (Werkstatt), nicht an den Chimären.

## Abgrenzung Kosmetik vs. Stats

| Element | Wo gekauft | Wirkung |
|---|---|---|
| Chimären-Teile | Chimären-Stall | kosmetisch |
| Charakter-Teile | Charakter-Erstellung | kosmetisch |
| Streitwagen | Streitwagen-Werkstatt | **Stats** (Geschwindigkeit, Gewicht) |
