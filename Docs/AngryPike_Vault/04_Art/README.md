## 📐 Dateiformate (verbindlich)

| Asset-Typ | Format | Max. Größe |
|---|---|---|
| Textures | `.png` / `.jpg` | 5 MB |
| UI-Elemente | `.png` / `.jpg` | 5 MB |
| SFX | `.wav` | — |
| Musik | `.mp3` / `.wav` | — |
| 3D-Models | `.fbx` | — |
| 3D-Animationen | `.fbx` | — |

> ⚠️ Dateien von s&box selbst werden **nicht** verändert.

---

## 📊 Polygon-Budgets

| Objekt-Typ | Max. Polygone |
|---|---|
| Environment Models | < 500 |
| Wichtige Objekte (z.B. Streitwagen) | < 2.000 |

## 🔄 2D-Asset Pipeline

```
Illustration (Procreate) → Colouring (Procreate) → Textures (Substance Painter) → Export (Codecks/Discord)
```

## 🔄 3D-Asset Pipeline

```
Modelling (3DS Max) → UV Unwrapping (3DS Max) → Texturing (Substance Painter) → Animation (s&box) → Import (s&box)
```
